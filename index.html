What is GRY?

GRY is a unitary css framework. WTF does that mean? it means that it's founded on the following principles:
  • mobile-first architecture
  • monoism
  • the golden ratio
  • utility classes

What's included with GRY?

In GRY, big-brain math has been done to calculate:
  • a 12-column grid system
  • a type scale based on the golden ratio (phi)
  • a 12-shade monochrome/greyscale color system
  • 3 distinct breakpoints
